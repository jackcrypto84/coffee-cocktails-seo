# Cocktail Image Prompt Template

Use this template when generating a hero image for a cocktail recipe.

Template:

```text
Photorealistic premium cocktail photography of {title}. Show the final drink exactly as served: {drinkColor}, in {glassware}, with {garnish}. Style cues: {styleTags}. Realistic lighting, realistic ice and liquid texture, accurate glassware, accurate garnish, clean composition, visually appetizing, no fantasy styling.
```

Rules:
- Match the actual recipe build, dilution style, and service temperature.
- Use the correct ice format for the drink style.
- Garnish should support the recipe, not decorate it randomly.
- Keep the liquid color believable for the ingredients used.
- Show realistic condensation, carbonation, and clarity when applicable.
- Avoid surreal smoke, neon liquids, impossible glassware, or AI-looking props.
